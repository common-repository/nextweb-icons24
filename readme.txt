This plugin allows use of 24x24 icons in admin menu and fixes CSS to match 24x24 size by adding few very simple lines to admin interface.

== Description ==
When creating custom post types and admin menu items sometimes you want to have more than 16x16 icon for menu items. Actually you can already use whatever size you want but it does not look god. This plugin allows use of 24x24 icons in admin menu and fixes CSS to match 24x24 size by adding few very simple lines to admin interface. It does not change existing menu items.
